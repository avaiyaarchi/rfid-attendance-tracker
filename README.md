# rfid-attendance-tracker
IoT-based Smart Attendance System using RFID, Arduino, Firebase, and Flutter

## 🔥 Features
- RFID-based attendance tracking
- Real-time Firebase database
- Mobile app interface
- Live data sync

## 🛠 Tech Stack
- Arduino
- Python
- Firebase
- Flutter
